# Brand Asset Inventory

This file is the compact source of truth for the bundled UNBOUNDX visual system.

## Brand DNA

- Keywords: hard-tech, dark immersion, precision, boundary-breaking, confident minimalism
- Brand mood: game-changer, engineering-grade, sharp, restrained, high-contrast
- Default posture: mostly dark, mostly quiet, with orange used as ignition rather than wallpaper

## Core Palette

| Role | Value | Use |
| --- | --- | --- |
| Aerospace Orange | `#F26122` | Primary accent, CTA, key highlights |
| Deep Void | `#050505` | Main background |
| Surface 1 | `#101012` | Cards, elevated panels |
| Surface 2 | `#1A1A1E` | Borders, separators, subtle structure |
| Surface 3 | `#25252A` | Hover states or deeper layering |
| Text Primary | `#FFFFFF` | Headlines, main copy on dark |
| Text Secondary | `#A0A0A5` | Support copy, metadata |
| Text Tertiary | `#606065` | Low-emphasis labels |

## Typography

- English / numbers: `Outfit`
- Chinese: `Noto Sans SC`
- Suggested weights:
  - `Outfit`: `300`, `400`, `600`, `800`
  - `Noto Sans SC`: `300`, `400`, `500`, `700`
- Typographic style:
  - big, geometric, high-tension headlines
  - small, crisp utility labels
  - strong contrast between title scale and metadata scale

## Style Rules To Preserve

- Dark first. `#050505` is the base, not `#000000`.
- Orange is intentional and sparse.
- Borders beat shadows.
- Geometry beats ornament.
- Negative space is part of the brand.
- Avoid generic “AI product” aesthetics such as purple glow, glass cards, and overloaded gradients.
